# 证据与安全边界

## 可以确认

- 本地程序可以读取进程可见的时区和环境变量。
- Claude Code 官方支持 `HTTP_PROXY` 和 `HTTPS_PROXY`。
- 网络服务通常能看到公网源 IP，并可识别其 ASN 和网络类型。
- TUN 与 macOS 系统代理工作在不同层面。
- Hysteria 2 使用 QUIC/UDP；端口、证书和参数不匹配会导致连接失败。
- VPS 通常是机房 IP；移动号码和蜂窝网络不等于家宽。

## 不能仅凭文章认定

- 固定数量的隐藏域名名单；
- 利用日期或 Unicode 偷传检测结果；
- 具体风控分数、队列或封禁阈值；
- 某个时区、VPS 品牌、住宅 IP、TLS 特征或运营商能保证安全；
- 删除本地文件会清除服务端记录。
- Anthropic 的每一封邮件都必然包含追踪像素。
- 指纹浏览器可以保证账号安全或不可识别。

需要判断平台当前行为时，优先查官方文档或可复现实物证据，并把逆向结果、经验和推测明确分开。

把本 Skill 用于网络可靠性、隐私、残留清理和自有服务器管理，不用于伪造所在地、身份或规避平台规则。
